CAPTCHA

Captcha(Completely Automated Public Turing test to tell Computers and Humans Apart) is used in attempts to prevent automated software from sending requests to a server. A good web scraping software should have some means for CAPTCHA recognition.

This test checks scraper's ability to:

  1. Recognize Character Captchas
  2. Scrape websites with Draggable Captchas
  3. Handle Drag & Drop Captchas

How it works:

  • Using the web scraper you need to get THE FORM WAS SUCCESSFULLY SUBMITTED for all three forms with captchas below
  • To achieve this in Character Captcha user needs to enter right characters into the text box
  • To achieve this in Draggable Captcha user needs to move the slider to the right
  • To achieve this in Drag & Drop Captcha user needs to move the proper icon to the circle
